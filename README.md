# C++ CLI Projects Collection

This repository contains a collection of **C++ command-line interface (CLI) projects** built to practice and demonstrate core C++ programming concepts, object-oriented design, file handling, and problem-solving skills.  
Each project is menu-driven and simulates real-world applications such as management systems and games.

---

## 📌 Project List

| Project Name | Description |
|--------------|-------------|
| **Banking_Management_System** | CLI-based banking application with account creation, deposit, withdrawal, and record management. |
| **Student_Management_System** | Manage student records (add, view, edit, delete) using file handling and menus. |
| **BuildOwnShell** | A simple custom shell implementation in C++ that executes basic commands. |
| **snakeGame** | Classic Snake game implemented in C++ using terminal-based controls. |
| **filehandling.cpp** | Demonstrates file input/output operations in C++. |
| **Other C++ Programs** | Additional practice programs for learning and experimentation. |
